## direct feed chat based on graffiti feed

Don't need to manually set up batchID, the script will look for it
Works with bee 1.16.1 dev version at the moment

### enter directory
```cmd
cd chat-feed-v1
```
### install dependencies
```js
npm install
```

### run direct.js script
```js
npm run direct
```
